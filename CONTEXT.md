직장인을 위한 캘린더 기반의 할일앱을 만들어줘
# 유저 
바쁜 직장인이 30초 안에 오늘의 할일을 확인하고 체크할 수 있어야 한다.
# 핵심 기능
- 유저는 월간 캘린더를 볼 수 있다
- 유저는 날짜를 클릭해서 그날의 할일 목록을 볼 수 있다
- 유저는 할일을 추가할 수 있다 (제목 + 우선순위 선택)
- 유저는 할일을 완료 체크할 수 있다
- 완료된 할일은 취소선과 함께 흐리게 표시된다
- 오늘 날짜는 항상 강조 표시된다
# 디자인
- 전체적인 분위기: 자라(ZARA) 공식 웹사이트처럼 고급스럽고 차가운 느낌
- 배경색: 순수한 검정 (#000000)
- 메인 포인트 색상: 형광 초록 (#39FF14)
- 버튼, 오늘 날짜, 체크박스에 사용
- 두번째 포인트 색상: 핫핑크 (#FF2D78) — 긴급 우선순위 할일에만 사용
